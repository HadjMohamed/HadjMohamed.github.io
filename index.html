My Portfolio

Explosion consequences prevision tool

My mission was to create a tool that could automatically generate reports based on user inputs, significantly reducing the time spent on manual data entry and report generation.

I developed a solution using Natural Language Processing (NLP) techniques to extract data from a 600-page physical workbook, leveraging Retrieval-Augmented Generation (RAG) methods. To clean and structure the data, I implemented an ETL pipeline orchestrated with Airflow, using Pandas for preprocessing.

All development was done under the supervision of four senior developers, following a robust CI/CD workflow. I automated the generation of reports and containerized the entire tool with Docker to ensure portability and reproducibility.

The tool was successfully adopted by the company, boosting the safety department's productivity by 60%.

Due to confidentiality agreements, I am unable to share the source code.

Chatbot based on personal data

When I first created this website, I felt something was missing: hIAdj. Beyond being proud of its name, this chatbot introduced me to text retrieval techniques and ML engineering.

Curious to see what a local chatbot could do with my own data, I built a lightweight RAG pipeline from scratch.

It retrieves the most relevant content from a ChromaDB vector database and feeds it to an open-source LLM (DeepSeek V3) to generate responses. If no useful document is found, it falls back to a custom prompt-only mode based on a pre-defined FAQ.

The whole system runs on a FastAPI backend, fully containerized with Docker and ready to deploy. A Makefile simplifies both local runs and cloud deploymentTerraform integration is on the roadmap.

While hIAdj may not match the efficiency of OpenAI models due to my limited resources, I focused on building a simple yet effective solution that puts my entire data engineering stack into action.

Reddit API Fake News

Born from a two-week challenge, the Reddit Fake News Detector is a data-driven tool designed to classify news headlines from Reddits r/worldnews as real or fake using NLP and machine learning.

We built a full pipelinefrom scraping headlines with the Reddit API to storing them in BigQuery, analyzing them with a fine-tuned BERT model, and displaying results through an interactive Streamlit app. Everything runs on a Python backend orchestrated with Airflow and exposed via FastAPI.

Updates were scheduled every 24 hours (no longer updated because of hosting costs), ensuring fresh data and live classification results. The goal? Help users visualize how content is classified and better understand the patterns behind online misinformation.
