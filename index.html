My Portfolio

Explosion consequences prevision tool

For safety flight department, I self-developed backend and user interface of a software able to predict explosions consequences.

The backend involved collecting experimental data from curves, processing the data by filtering out irrelevant points, and loading it into a database.

Through an OOP Python-based UI integrated with a database, the tool automatically generated reports based on user inputs.

This project was particularly challenging due to working within a secured Python environment. However, collaborating with senior developers enhanced my comfort with data manipulation and efficient programming practices.

The tool was successfully adopted by the company, boosting the safety department's productivity by 60%.

Due to confidentiality agreements, I am unable to share the source code.
