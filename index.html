My Portfolio

Explosion consequences prevision tool

For safety flight department, I self-developed backend and user interface of a software able to predict explosions consequences.
The backend has consisted in collecting experimental data from curves, processing data by removing unrelevant points and load it into a database.
Then, through the User interface and a Binary Decision Diagram the tool automatically generated a report corresponding to explosions parameters.
This project was really challenging due to working in a securised Python environment but working with senior developers allowed me to more comfortable with data manipulation and effective programmation.
My tool has been added to the company rising up by 60% safety department productivity.
In order to respect my confidential agreement source code can unfortunately not be given.

Chatbot based on personal data

I honestly was not satisfied of this website when I created it, something was missing: hIAadj. On top of being very proud of its name, this chatbot introduced myself to some of text retrieval techniques (cosine similarity, vector database, MMR). I was not looking for an existing AI implemented through API prompting, I was really excited about creating myself the text generation it is what I did ! Of course the chatbot may be not efficient as an OpenAI model, due to my available ressources, I focused on a simple way to answering questions without being dependant of an existing public model.
